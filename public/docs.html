Documentação da API

Guia completo para integrar com a nossa API mágica!

GET /api/status Status

Verifica a saúde e a disponibilidade da API. Ideal para monitoramento.

Resposta de Sucesso (200 OK )

Exemplo de Resposta 
{
  "status": "online",
  "message": "API Qode está funcionando...",
  "version": "1.1.0",
  "timestamp": 1678886400000
}
GET /api/generate Principal

O endpoint principal para gerar a imagem de um QR Code.

Parâmetros (Query )

Parâmetro Tipo Obrigatório Descrição
text string Sim O conteúdo a ser codificado no QR Code.
dark hex Não Cor dos pontos. Padrão: #000000.
light hex Não Cor do fundo. Padrão: #FFFFFF.

Exemplo de Requisição

Exemplo de URL 
/api/generate?text=AmoGatinhosFofos

Resposta de Sucesso (200 OK)

Exemplo de Resposta 
{
  "success": true,
  "result": {
    "dataUrl": "data:image/png;base64,iVBORw0KG...",
    "format": "png"
  }
}
